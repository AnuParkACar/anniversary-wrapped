# Place your background music here

Add a music file named `background.mp3` to this folder.

## Recommended sources for royalty-free music:
- [Pixabay Music](https://pixabay.com/music/) - Free, no attribution required
- [Free Music Archive](https://freemusicarchive.org/) - Various licenses
- [YouTube Audio Library](https://studio.youtube.com/channel/audio) - Free for creators

## Tips:
- Choose something romantic and meaningful to your relationship
- Keep file size reasonable (2-5MB is good)
- MP3 format works best for browser compatibility

## Custom songs:
If you want to use a specific song that's meaningful to you:
1. Make sure you have rights to use it (personal use should be fine)
2. Convert to MP3 if needed
3. Name it `background.mp3`
